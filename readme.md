# Vtipy
A jokes database

##Instalation

###Linux

```javascript

    sudo apt-get install npm
```
##Mac

1) open console and clone repo 

```javascript

    git clone https://github.com/matkuscz/vtipy.git
    cd vtipy
```

2)
```javascript

    brew update
    brew postinstall

    npm install

    npm install nodemon

    npm install --global gulp-cli

    gulp
```

In a new tab run app

```javascript

    npm run watch
```
