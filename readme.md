# Vtipy
A jokes database

##Instalation

###Linux

```javascript

    sudo apt-get install npm
```
##Mac

```javascript

    brew update
    brew postinstall
```

//
```javascript

    git clone https://github.com/matkuscz/vtipy.git
    cd vtipy

//

```javascript

    npm install

    npm install nodemon

    npm install --global gulp-cli

    gulp
```

In a new tab run app

```javascript

    npm run watch
```
